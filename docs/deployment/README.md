# Narrowcasting Deployment Documentation

This folder contains deployment and production operation guides.

## Documents

| Document | Purpose |
| --- | --- |
| [PRODUCTION_DEPLOYMENT.md](PRODUCTION_DEPLOYMENT.md) | Raspberry Pi appliance mode, production build, systemd services, kiosk launch, and boot recovery. |
| [INSTALLATION.md](INSTALLATION.md) | Unified interactive installer and direct server/player install scripts for Raspberry Pi/Linux deployments. |

Deployment documentation describes how to install and operate Narrowcasting in production. It should not define product behaviour or technical architecture.
